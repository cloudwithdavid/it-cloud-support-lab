# Ticket Lifecycle

## Standard Lifecycle

- New
- Triage
- In Progress
- Pending
- Resolved
- Closed

## Lifecycle Intent

### New
Issue has been created but not yet assessed.

### Triage
Problem is being scoped, classified, and evaluated.

### In Progress
Checks, troubleshooting, or remediation are actively being performed.

### Pending
Waiting on user response, external dependency, or follow-up action.

### Resolved
A fix or supported outcome has been reached and verified.

### Closed
Work is complete and the ticket record is finalized.

## Escalation

If the issue cannot be resolved within the expected support boundary, the ticket should be escalated with:

- problem summary
- impact
- relevant checks performed
- findings
- current status
- reason for escalation
