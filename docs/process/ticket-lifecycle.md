# Ticket Lifecycle

## Lifecycle Used in This Lab

- New
- In Progress
- Resolved
- Escalated
- Closed

## Status Meanings

### New
Issue has been created but not yet actively worked. (Also the initial triage pool.)

### In Progress
Checks, troubleshooting, or remediation are actively being performed.

### Resolved
A fix or supported outcome has been reached and verified.

### Escalated
The issue was judged to be beyond the current support boundary.

### Closed
Work is complete and the ticket record is finalized.
